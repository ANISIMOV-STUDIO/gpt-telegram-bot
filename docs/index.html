GPT TELEGRAM BOT

ChatGPT-powered assistant with context memory

🤖 Features

  • ChatGPT integration via OpenAI API
  • Per-user conversation context memory
  • HTTP/SOCKS5 proxy support for restricted regions
  • Access control with user whitelist
  • Automatic context cleanup
  • GitHub Actions CI/CD ready

🚀 Quick Start

git clone https://github.com/ANISIMOV-STUDIO/gpt-telegram-bot.git
cd gpt-telegram-bot
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp env.example .env # Configure your tokens
python run.py
